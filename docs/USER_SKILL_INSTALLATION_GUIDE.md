> **版本**: 1.0  
> **最后更新**: 2026-03-23

## 📚 快速入门

本指南帮助你快速安装和管理 OpenClaw Skills(技能)。整个过程只需 3-5 步即可完成。

---

## 一、从技能市场安装 Skills(推荐方式)

**适用场景**: 安装社区提供的公共技能,如飞书、GitHub、天气查询等

### 步骤 1: 找到技能管理入口

**界面位置**:
- 启动 OpenClaw Desktop 应用
- 在左侧导航栏底部,找到 **Skills** 图标(📚 书本图标)
- 点击进入技能管理页面

```
左侧导航栏
├─ 🏠 首页
├─ 💬 对话
├─ ⚙️ 设置
└─ 📚 Skills  ← 点击这里
```

### 步骤 2: 切换到"市场搜索"

**界面位置**:
- 进入技能管理页面后,顶部有 3 个标签页:
  - [本地技能] [市场搜索] [插件]
- 点击 **"市场搜索"** 标签

### 步骤 3: 搜索技能

**操作方法**:
1. 在页面顶部的搜索框中输入关键词
   - 例如: `feishu`、`github`、`weather`
2. 点击搜索框右侧的 **"搜索"** 按钮(或按 Enter 键)
3. 等待搜索结果展示(通常 2-5 秒)

**界面说明**:
```
[搜索框: 输入关键词... ] [搜索按钮]

搜索结果区域:
┌─────────────────┐  ┌─────────────────┐
│  技能 A         │  │  技能 B         │
│  描述信息...    │  │  描述信息...    │
│  [安装] 按钮    │  │  [安装] 按钮    │
└─────────────────┘  └─────────────────┘
```

### 步骤 4: 安装技能

**操作方法**:
1. 找到你需要的技能卡片
2. 点击卡片上的 **"安装"** 按钮
3. 等待安装完成
   - 按钮会显示 "安装中..."
   - 安装成功后按钮变为 "已安装"
   - 页面右上角会弹出绿色成功提示

**进度反馈**:
- 安装中: 按钮显示加载动画
- 成功: 绿色提示 "✓ 技能安装成功"
- 失败: 红色提示 "安装失败,请查看日志"

### 步骤 5: 验证安装

**操作方法**:
1. 自动切换到"本地技能"标签
2. 查看技能列表中是否出现你刚安装的技能
3. 技能状态应显示为"已安装"

## 二、管理已安装的 Skills

### 如何查看已安装技能

**界面位置**:
- 进入 Skills 页面
- 默认显示"本地技能"标签
- 所有已安装技能以卡片形式展示

**界面说明**:
```
本地技能列表:
┌─────────────────────────────┐
│ 🔥 feishu-bitable           │
│ 飞书多维表格管理技能         │
│ 状态: 已安装 ✅             │
│ [编辑] [删除]                │
└─────────────────────────────┘
```

### 如何查看技能详情

**操作方法**:
1. 在技能列表中,点击任意技能卡片
2. 右侧会滑出技能详情面板
3. 查看技能的完整信息:
   - 技能名称和描述
   - 版本号和作者
   - 技能功能说明
   - 依赖项(如有)

### 如何禁用/启用技能

**操作方法**:
1. 点击技能卡片,打开详情面板
2. 在详情面板中找到开关按钮
   - 开关在右侧 = 技能已启用
   - 开关在左侧 = 技能已禁用
3. 点击开关切换状态
4. 等待 1-2 秒,状态生效

**界面说明**:
```
技能详情面板:
┌──────────────────────────┐
│ 技能名称: feishu-bitable │
│ 描述: ...                │
│                          │
│ 状态:                    │
│   [○] 禁用  [●] 启用   │ ← 点击切换
│                          │
│ 版本: 1.0.0              │
│ 作者: OpenClaw Team      │
└──────────────────────────┘
```

### 如何卸载技能

**操作方法**:
1. 在技能列表中找到要卸载的技能
2. 点击技能卡片
3. 在右侧详情面板中,点击 **"删除"** 按钮
4. 确认删除操作
5. 技能从列表中移除

**注意**: 卸载后技能将无法使用,如需重新使用请重新安装

### 如何更新技能

**当有新版本可用时**:
1. 技能状态会显示 "可更新" (黄色)
2. 点击技能卡片
3. 在详情面板中点击 **"更新"** 按钮
4. 等待更新完成

## 三、从本地文件安装 Skills(高级)

**适用场景**: 你有开发者提供的技能包文件(如 .zip 或文件夹)

### 步骤 1: 找到安装入口

**界面位置**:
- 进入 Skills 页面 → "本地技能" 标签
- 在页面顶部工具栏,点击 **"创建技能"** 按钮

### 步骤 2: 选择安装来源

**弹窗界面**:
- 会弹出"创建技能"对话框
- 选择 **"从本地文件安装"** 选项

### 步骤 3: 选择文件

**操作方法**:
1. 点击 **"选择文件"** 按钮
2. 在文件选择器中:
   - 选择技能包文件(.zip)或技能文件夹
   - 确保文件中包含 `SKILL.md` 文件
3. 点击 **"打开"**

### 步骤 4: 确认安装

**操作方法**:
1. 系统会自动检测文件内容
2. 确认技能信息无误后
3. 点击 **"安装"** 按钮
4. 等待安装完成

**进度反馈**:
- 解压中: 显示解压进度
- 验证中: 检查文件完整性
- 安装中: 复制文件到系统目录
- 完成: 显示成功提示

## 四、安装后如何使用技能

### 在对话中触发技能

**操作方法**:
1. 进入首页或对话页面
2. 在聊天框中输入相关问题
3. Agent 会自动识别并调用相应技能

**示例**:
```
你: 查询飞书多维表格中的数据
    ↓
Agent: [调用 feishu-bitable 技能]
    ↓
Agent: 已找到 10 条记录...
```

### 检查技能是否正常工作

**方法 1: 运行诊断**
1. 进入 Skills 页面
2. 点击顶部 **"诊断"** 按钮
3. 查看诊断结果:
   - ✅ 绿色 = 正常
   - ⚠️ 黄色 = 有警告
   - ❌ 红色 = 有错误

**方法 2: 测试对话**
1. 打开与 Agent 的对话
2. 输入一个需要该技能的问题
3. 观察 Agent 是否成功调用技能

## 五、常见问题

### Q1: 安装按钮是灰色的,无法点击?

**A**: 可能原因:
- 技能已安装(按钮显示"已安装")
- 网络连接中断
- 技能不兼容当前系统版本

**解决方法**: 刷新页面或检查网络连接

### Q2: 安装过程中卡住不动?

**A**:
- 等待最多 30 秒
- 如仍无响应,关闭应用后重试
- 或通过日志查看详细错误信息(设置 → 日志中心)

### Q3: 安装成功但对话中无法使用?

**A**: 检查以下项:
1. 技能是否为"启用"状态(看详情面板中的开关)
2. 是否缺少依赖(详情面板会显示缺失的依赖)
3. 运行诊断,查看是否有错误

### Q4: 如何知道哪些技能是我自己安装的?

**A**:
- 在技能列表中,自定义技能会显示 **"自定义"** 标签
- 这些技能支持编辑和删除操作
- 内置技能无法删除

### Q5: 安装技能时提示"权限不足"?

**A**:
- 请以管理员身份运行 OpenClaw Desktop
- 或在系统设置中授予应用相关权限
- Windows 用户: 右键应用图标 → "以管理员身份运行"

## 七、快速操作流程图

```
安装新技能:
启动应用 → 点击 Skills → 点击"市场搜索" 
→ 输入关键词 → 点击"搜索" → 点击"安装" → 完成

管理已安装技能:
点击 Skills → 点击"本地技能" → 选择技能 
→ 查看详情/编辑/禁用/删除

诊断技能问题:
点击 Skills → 点击"诊断" → 查看结果 → 按提示修复
```

## 💡 使用小贴士

1. **首次使用**: 建议先安装 1-2 个常用技能,熟悉操作流程
2. **定期更新**: 技能会持续更新,建议定期检查更新
3. **善用搜索**: 技能市场支持模糊搜索,尝试不同关键词
4. **查看描述**: 安装前仔细阅读技能描述,确保符合需求
5. **遇到问题**: 先运行诊断,获取详细错误信息

# OpenClaw Desktop 设计系统（Apple风格）
---
## 核心设计原则
1. **极简轻量化**：减少视觉噪点，优先复用macOS原生组件，无冗余装饰
2. **磨砂玻璃效果**：全局使用NSVisualEffectView材质，适配系统明暗模式
3. **高对比度**：文本与背景对比度≥4.5:1，符合WCAG AA无障碍标准
4. **原生一致性**：100%遵循Apple Human Interface Guidelines，与系统体验无缝衔接
5. **主色调适配**：所有设计元素适配指定科技蓝（#00B4FF）、科技绿（#00E08E）主色调

---
## 1. 配色系统
### 1.1 品牌主色
| 色值 | 名称 | 用途 |
|------|------|------|
| `#00B4FF` | 科技蓝 | 主按钮、链接、选中状态、焦点边框、进度条主色 |
| `#00E08E` | 科技绿 | 成功状态、操作完成提示、正向反馈、在线状态 |

### 1.2 功能色（复用macOS原生系统色）
| 色值 | 名称 | 用途 |
|------|------|------|
| `#FF9500` | 警告橙 | 告警提示、警告状态、需注意操作 |
| `#FF3B30` | 错误红 | 错误提示、失败状态、危险操作 |
| `#AF52DE` | 辅助紫 | 标记、特殊状态提示 |

### 1.3 中性色（自动适配明暗模式）
| 用途 | 浅色模式色值 | 深色模式色值 |
|------|--------------|--------------|
| 窗口背景（磨砂玻璃基底） | `#FFFFFF` | `#1E1E1E` |
| 卡片/模块背景 | `#F5F5F7` | `#2C2C2E` |
| 分割线/边框 | `#E5E5EA` | `#3A3A3C` |
| 次要文本/提示 | `#8E8E93` | `#98989F` |
| 主要文本/标题 | `#1C1C1E` | `#F5F5F7` |

### 1.4 磨砂玻璃效果参数
| 场景 | 材质 | 透明度 | 圆角 |
|------|------|--------|------|
| 主窗口 | `NSVisualEffectView.Material.underWindowBackground` | 85%（浅色）/90%（深色） | 12pt |
| 卡片/弹窗 | `NSVisualEffectView.Material.popover` | 90% | 8pt |
| 组件（按钮/输入框） | 纯色/半透明 | - | 6pt |
| 图标按钮/小控件 | 半透明背景 | - | 4pt |

| 层级 | 字号（pt） | 字重 | 用途 |
|------|------------|------|------|
| 大标题 | 24 | Bold | 窗口标题、启动页标题 |

| 正文 | 13 | Regular | 说明文本、内容主体、配置项值 |
| 辅助文本 | 11 | Regular | 提示信息、次要说明、版本号 |

## 3. 组件规范
### 3.1 按钮
| 类型 | 样式 | 交互效果 |
|------|------|----------|
| 主按钮 | 背景科技蓝#00B4FF，文字白色，内边距10pt 16pt，圆角6pt | Hover：亮度+10%；点击：亮度-10%，轻微缩放0.98倍 |
| 次要按钮 | 透明背景，边框1pt中性色分割线，文字主文本色，内边距10pt 16pt，圆角6pt | Hover：背景填充卡片背景色 |
| 图标按钮 | 24x24pt，透明背景，图标使用主文本色 | Hover：背景填充卡片背景色，圆角4pt |

### 3.2 开关（Toggle）
- 开启状态：背景科技蓝#00B4FF，完全对齐macOS原生开关样式
- 关闭状态：背景中性色分割线
- 交互：滑动动画时长200ms，符合系统动效

### 3.3 输入框
- 边框1pt中性色分割线，圆角6pt，内边距8pt
- 焦点状态：边框科技蓝#00B4FF，阴影`0 0 0 3pt rgba(0, 180, 255, 0.2)`
- 禁用状态：透明度50%

### 3.4 侧边栏（设置页导航）
- 宽度220pt，磨砂玻璃背景，内边距16pt 0
- 选中项：背景`rgba(0, 180, 255, 0.1)`，文字科技蓝#00B4FF，左侧2pt高亮边框
- 未选中项：文字主文本色，hover背景填充卡片背景色
- 分组标题：辅助文本大小，字重Medium，上边距24pt，下边距8pt

---
## 4. 交互规范
1. **动画**：所有过渡动画时长150ms-300ms，使用Apple原生缓动曲线，无突兀动效
2. **反馈**：所有可点击元素点击时有轻微状态反馈（颜色变化/缩放）
3. **导航**：侧边栏点击切换页面，使用淡入淡出过渡，无刷新跳转
4. **窗口行为**：完全符合macOS窗口规范，支持缩放、最小化、关闭、全屏，支持拖放调整大小
5. **响应式**：窗口最小宽度800pt，最小高度600pt，支持任意比例缩放

# OpenClaw Desktop 设置页设计（共10个子页面）
---
## 页面通用布局
- 左侧固定220pt侧边栏导航，列出10个设置项，符合设计系统侧边栏规范
- 右侧内容区宽度自适应，内边距32pt，顶部为页面标题（标题1样式）
- 内容区按功能分组，每组之间间隔24pt，分组标题使用标题2样式，下方为设置项列表
- 所有设置项左右对齐，标签左对齐，控件右对齐，辅助说明文本位于标签下方（辅助文本样式）

---
## 1. General（通用设置）
### 包含功能：
| 设置项 | 组件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 开机自启动 | 开关 | 关闭 | 系统启动时自动运行OpenClaw |
| 启动时最小化到状态栏 | 开关 | 关闭 | 启动后不显示主窗口，仅在状态栏显示图标 |
| 外观模式 | 单选组 | 跟随系统 | 选项：跟随系统/强制浅色/强制深色 |
| 启用磨砂玻璃效果 | 开关 | 开启 | 关闭后使用纯色背景，降低性能消耗 |
| 语言 | 下拉选择 | 跟随系统 | 支持：简体中文/English/日本語 |
| 状态栏图标显示 | 开关 | 开启 | 关闭后状态栏不显示图标，需通过快捷键唤醒 |
| 点击状态栏图标行为 | 下拉选择 | 打开主窗口 | 选项：打开主窗口/显示快捷菜单 |

### 布局：
分为「启动设置」「外观设置」「语言与状态栏」3个分组，每个分组垂直排列设置项。

---
## 2. Channels（渠道设置）
### 包含功能：
- 顶部操作区：搜索框 + 「添加渠道」主按钮
- 已绑定渠道列表：卡片式布局，每个卡片包含：
  - 渠道图标 + 渠道名称
  - 绑定状态标签（绿色「已绑定」/橙色「待授权」/红色「已失效」）
  - 操作按钮：配置/解绑
- 绑定新渠道弹窗：展示所有支持的渠道（飞书/企业微信/钉钉/Slack/Discord等），点击后跳转授权流程

### 布局：
卡片宽度自适应，每行最多显示3个卡片，卡片带磨砂玻璃效果，圆角8pt，内边距16pt，hover时有轻微上浮效果。

## 3. Voice Wake（语音唤醒设置）
### 包含功能：
| 设置项 | 组件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 语音唤醒总开关 | 开关 | 关闭 | 开启后支持语音唤醒助手 |
| 自定义唤醒词 | 输入框 + 测试按钮 | "小Claw" | 最多支持8个汉字/16个字符 |
| 唤醒灵敏度 | 滑块 | 5 | 范围1-10，数值越高越灵敏，误触率越高 |
| 唤醒声音反馈 | 开关 | 开启 | 唤醒成功时播放提示音 |
| 唤醒弹窗提示 | 开关 | 开启 | 唤醒成功时显示提示弹窗 |
| 唤醒后自动聚焦输入框 | 开关 | 开启 | 唤醒后直接可以输入指令 |
| 识别引擎 | 下拉选择 | 本地模型 | 选项：本地离线模型/云端高精度模型 |

### 布局：
分为「基础设置」「唤醒反馈」「高级设置」3个分组，滑块实时显示当前数值，测试按钮点击后可直接测试唤醒效果。

---
## 4. Config（核心配置）
### 包含功能：
| 设置项 | 组件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 工作目录 | 输入框 + 选择按钮 | `~/.openclaw/workspace` | 点击选择按钮打开系统文件选择器 |
| 代理设置 | 下拉选择 + 输入组 | 无代理 | 选项：无代理/HTTP代理/SOCKS5代理，选择后显示地址/端口输入框 |
| 日志级别 | 下拉选择 | Info | 选项：Debug/Info/Warn/Error |
| 自动检查更新 | 开关 | 开启 | 启动时自动检查新版本 |
| 更新渠道 | 下拉选择 | 稳定版 | 选项：稳定版/开发版/每日构建版 |

### 布局：
分为「工作空间」「网络设置」「更新设置」3个分组，代理相关输入框仅在选择非无代理选项时显示。

---
## 5. Extensions（扩展管理）
### 包含功能：
- 顶部操作区：搜索框 + 「浏览扩展市场」主按钮 + 「从本地安装」次要按钮
- 已安装扩展列表：卡片式布局，每个卡片包含：
  - 扩展图标 + 扩展名称 + 版本号 + 作者
  - 启用/禁用开关
  - 操作按钮：配置/卸载
- 扩展市场弹窗：展示推荐扩展、分类筛选、搜索功能，支持一键安装

### 布局：
卡片宽度自适应，每行最多显示2个卡片，支持按名称/安装时间排序，禁用的扩展透明度为60%。

## 6. Notifications（通知设置）
### 包含功能：
| 设置项 | 组件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 全局通知开关 | 开关 | 开启 | 关闭后所有通知都不显示 |
| 任务完成通知 | 开关 | 开启 | 任务执行完成时发送通知 |
| 消息提醒通知 | 开关 | 开启 | 收到新消息时发送通知 |
| 系统告警通知 | 开关 | 开启 | 出现异常/告警时发送通知 |
| 更新提示通知 | 开关 | 开启 | 有新版本时发送通知 |
| 横幅通知 | 开关 | 开启 | 右上角显示临时横幅通知 |
| 保留到通知中心 | 开关 | 开启 | 通知在系统通知中心保留 |
| 通知声音 | 开关 | 开启 | 通知播放提示音 |
| 勿扰模式 | 开关 | 关闭 | 开启后所有通知静音不弹窗 |
| 勿扰时间段 | 时间选择组 | 22:00 - 07:00 | 开启后该时间段自动进入勿扰模式 |

### 布局：

## 8. About（关于页面）
### 包含功能：
- 顶部展示区（居中）：
  - OpenClaw logo（128x128pt）
  - 应用名称（大标题样式）
  - 版本号 + 构建号 + 更新时间（辅助文本样式）
- 快捷入口区（网格布局）：
  - 官方网站按钮
  - 文档中心按钮
  - GitHub仓库按钮
  - 反馈问题按钮
  - 检查更新按钮
- 底部版权区：
  - 版权声明文本
  - 开源协议链接
  - 隐私政策链接

### 布局：
整体居中对齐，快捷入口按钮为次要按钮样式，每行显示3个，检查更新按钮点击后显示更新状态提示。

|--------|----------|--------|------|

## 10. Logs（日志查看）
### 包含功能：
- 顶部筛选操作栏：
  - 日志级别筛选下拉（全部/Debug/Info/Warn/Error）
  - 时间范围筛选下拉（今天/最近7天/全部）
  - 关键词搜索框
  - 操作按钮：刷新/清空/导出
- 日志内容区：
  - 可滚动文本区域，占满剩余空间
  - 使用等宽字体（SF Mono）显示日志
  - 不同级别日志使用不同颜色标记：Debug灰色、Info默认、Warn橙色、Error红色

### 布局：
筛选栏位于顶部，日志内容区自适应高度，支持滚动查看历史日志，导出按钮点击后将日志保存为文本文件。

## 1. Product Overview

OpenClaw Desktop is a desktop management tool for the multi-agent AI system. You can easily manage AI agents, configure the system, monitor running status, and schedule automated tasks through a visual interface without complex command-line operations.

---

## 2. Core Feature Modules

### 2.1 Dashboard

The system homepage for quickly viewing overall running status and performing frequent operations. It is the core entry point for using OpenClaw.

### 2.2 Agent Management

Manage all AI agents, including creating, editing, starting, stopping, and configuring agent parameters.

### 2.3 Task Management

Create and manage scheduled tasks (cron jobs) to automate agent operations.

### 2.4 Skills Management

Browse, install, and manage skills that extend agent capabilities.

### 2.5 System Settings

Configure application settings including theme, language, gateway connection, and advanced options.

### 2.6 Session Management

View and manage conversation sessions between users and agents.

### 2.7 Instance Management

Monitor and control running OpenClaw instances.

### 2.8 Log Management

View application and system logs for debugging and monitoring.

---

## 3. Quick Start Guide

1. Launch OpenClaw Desktop
2. Complete the setup wizard to connect to your OpenClaw installation
3. Navigate to the Dashboard to view system status
4. Use the sidebar to access different feature modules

---

## 4. FAQ

**Q: How do I connect to a remote OpenClaw instance?**
A: Go to Settings and configure the gateway connection with your remote host address.

**Q: How do I change the language?**
A: Go to Settings > General and select your preferred language.

**Q: How do I switch between light and dark themes?**
A: Use the theme toggle buttons at the bottom of the sidebar.

# OpenClaw Desktop

OpenClaw Desktop is an Electron-based desktop application that serves as a management dashboard for the OpenClaw multi-agent AI system. It allows users to monitor and control agents, sessions, tasks, logs, instances, and skills running on a local or remote OpenClaw runtime.

The app includes a first-run setup flow that guides users through connecting to a local or remote OpenClaw installation. Once connected, users interact with the system through a sidebar-navigated UI with pages for each major domain (Dashboard, Agents, Sessions, Tasks, Logs, Instances, Skills, Settings).

The UI supports light/dark/system themes, internationalization (currently Chinese and English), and a custom frameless window with a hidden-inset title bar on macOS.
